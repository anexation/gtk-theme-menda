gtk-theme-menda
===============

New rebase for manjaro's gtk theme menda off of Vertex by horst3180

Name: Menda

Summary: A light and dark theme with a mix of green and light/dark grey colors.

URL: http://manjaro.org

Licensed as GPLv3.

The Menda theme includes:
- GTK2 theme
- GTK3 theme
- Gnome shell
- Cinnamon Theme
- Metacity theme
- Xfwm4 theme
- Openbox Theme
